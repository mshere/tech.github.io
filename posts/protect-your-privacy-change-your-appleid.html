AppleId换区

Posted on 2018-01-17 | In tools | | 本文总阅读量 次
Words count in article: 3,048 | Reading time ≈ 11

为了保护自身AppleId数据的隐私与安全，强烈建议更换AppleId区域。

版权归作者所有，任何形式转载请联系作者。

作者：Azeril（来自豆瓣）

来源：https://www.douban.com/note/653557027/

因由

因为国内政策问题（都是套路），Apple 中国区的 iCloud 云存储服务将在今年 3 月开始正式落地于大陆，通过地处贵州的「云上贵州」公司运作相关事宜，并接受大陆政府的监管。当然，如果关注过这几年的种种事端，你会发现这仅是一个本就木已成舟的事实的表面化而已。

选择一个智能机系统，很大程度上是选择一个生态。而中国区的网络生态是如何的呢，对于 iPhone 来说，是购买时附带更高的税收，使用时更不稳定的云服务（这是 Apple 的锅），以及出于一以贯之的政策，经常性的封锁与压制。远的不说，17 年秋末几天之中，数百款 VPN 类工具相继被下架，冬天时，国外的新闻订阅类应用也步其后尘。尤其是前者带来的附带杀伤力，让国内连接国际互联网原本拥堵的带宽也一下子畅通了…而与此同时也意味着，更多的人不得已无法再使用 Google、Youtube、Twitter 等知名站点的优良服务。而此番 iCloud 云存储服务落地中国，则无非是雪上加霜罢了。

其实先前除开主力的国区 Apple ID，本就已有一个备用的美区帐号，用于下载往日并未在国内上架的应用。本来，所购应用都在主力帐号上，得过且过也无非如此，毕竟还有一个帐号可以平衡。而眼下已经愈加受制的使用体验之外，又来一个更明朗化的数据安全问题，就多少无法再接受了。

因而，出于一些小小的执念，最终还是决定将主力 Apple ID 帐号及相关的服务（包括牵涉到的 iCloud 存储服务）由国区转为美区，这样的话日常使用 Apple 系列产品时所产生的相关数据即不会存储在大陆的服务器中。

当然，由于先前相关的服务牵涉不少，转区这一件原本预想并不复杂的事情还是生出不少波折，也费了一点劲才算搞定。完成之后，将踩的几个坑也顺带给填上，有此番需要的也可以直接绕过。是为一记。

转区操作之前

要转区得先处理好与自己的 Apple ID 相关的一些服务：

  • 退订应用内订阅服务
  • 取消 iCloud 存储空间付费服务
  • 关闭家庭共享账户
  • 关闭双重验证
  • 用掉账户余款，进行这些操作之前记得先连接 iTunes 为 iOS 设备进行备份，以防万一。

接下来说一下具体的操作步骤，如果没有添加相关服务，可跳过无关部分继续往下。

退订订阅

当前 Apple ID 下订阅的所有会自动续订的服务都需要关闭掉。主要分两类，一类是 Apple 自家的 Apple Music, Apple New 等；另一类是第三方应用中的内容或服务订阅。

执行取消订阅的操作，有两个主要渠道。

一、是在基于 iOS 系统的 iPhone 或 iPad 及 iPod Touch 等设备上操作完成。

  1. 点击「设置」-顶部的个人账户「名字」-「iTunes Store 与 Apple Store」；
  2. 点击顶部「Apple ID」一栏；
  3. 在跳出的小窗口中，选择「查看 Apple ID」（View Apple ID），并验证通过；
  4. 点击「订阅」，进入订阅管理列表查看和进行取消操作。

二、是基于 macOS 和 Windows 系统下的 iTunes 来操作：

  1. 打开 iTunes，并登录个人的 Apple ID；
  2. 顶部菜单栏，依次点击「账户」-「查看我的账户」，并验证通过；
  3. 在加载的账户信息页面，查看「设置」一栏下的「订阅」选项，如果有历史订阅，则会出现订阅的条目、订阅数及「管理」按钮，不然则不会出现这一选项；
  4. 点击「管理」跳转订阅列表进行取消操作。

取消 iCloud 存储空间付费服务

官方说法是叫「降级 iCloud 存储空间」。默认情况下注册和使用 Apple ID 会自带 5G 的免费存储空间。如果付费订阅则可以升级到更多。

初期操作时可略过，无法正常换区并收到提醒时，请先关闭订阅的 iCloud 存储服务。

降级的方法简单综述一下：

  • 在 iOS 设备上：

「设置」-「名字」-「iCloud」-「管理存储空间」（Manage Storage）——「更改存储空间方案」——点击「降级选项」，输入密码确认——选取免费方案，并点击「完成」；

  • 在 macOS 上：

「系统偏好设置」-「iCloud」——点击窗口右下角「管理」——点击新窗口左上角「更改存储空间方案」——点击「降级选项」，输入密码确认——选取免费方案，并点击完成。

关闭家庭共享账户

因为涉及到 Apple ID 换区，不同国家地区之下的帐号无法互通，因而如果有开启家庭共享群组的话，在换区操作前需要先关闭掉。途径与退订操作类似：

  • 在 iOS 设备上：

「设置」-「家庭共享」——点击自己名字——「停止家人共享」；

  • 在 macODS 上：

「系统偏好设置」-「iCloud」-「管理家人共享」——「点击自己名字」——「移除」。

关闭双重验证

有可能会遇到提示，必须关闭才能继续换区操作，有的话先关闭，换区完成后再重新开启：

  1. 登录「管理您的 Apple ID」页面：https://appleid.apple.com/
  2. 在「安全」一栏对「双重认证」选项下点击关闭。

    用掉帐户余款

通过 App Store / iTunes Store 礼品卡、银联支付等渠道充值到 Apple ID 的余额，需要归零。可以通过在 App Store 中购买应用，或解锁应用内购等消耗掉。

转区操作

必要准备

进行转区时，涉及到以下的一些内容：

  • macOS Sierra 及以下版本操作系统（Windows 下状况不明） / 任意 iOS 设备；
  • 支持 App Store 绑定的双币银行卡（非必须，有更好…当然要是有就没这么多事了其实）；
  • 一个美国地址；
  • 一个服务器 IP 位于美区的梯子；最为重要的转区操作在此处再详细提及一下：

iOS：

  1. 点击「设置」-顶部个人账户「名字」-「iTunes Store 与 App Store」；
  2. 点击顶部「Apple ID」一栏；
  3. 在跳出的小窗口中，选择「查看 Apple ID」（View Apple ID），并验证通过；
  4. 点击「国家」-「下一步」-在新的条款页查看并确认 Apple 的相关条款与隐私政策；
  5. 付款方式选选项「None」，邮寄账单地址的选项，输入已准备的地址；
  6. 点击「下一步」完成。macOS：iTunes，登录帐号——顶部菜单栏「帐号」—「查看我的账户」-「切换国家/地区」 后续的操作与 iOS 类似，不赘述。

可能遇到的问题及解决方案

测试发现，使用 macOS High Sierra 下的 iTunes 进行转区，修改个人信息时，必须为 App了 ID 选定一个银行卡绑定作为支付渠道，而美区或双币银行卡的持有者与有转美区需求的人群两个条件交集并不多，因而这个限制条件就很让人犯难。

这个问题，可以通过在 Sierra 及以下版本中，使用 iTunes，或者是通过任意的 iOS 设备进行相应操作来规避。当然，如果还是遇到问题，请在操作 iTunes 时全程使用美区 IP 的网络连接稳定的梯子即可。

在理想状态下，这时支付渠道的选项会显示一个「None」选项，选择此选项，相当于我们可跳过难搞的银行卡绑定操作。那样的话，接下来只需填写伪美区地址和一个美区手机号——最好是个人持有的真实美国号码，当然假的问题也不大，具体下一部分说。

关于美区地址，可以选择自行使用 Google Maps 定位信息或地址/身份生成器生成一个美国的伪地址填进去。放一个生成器（或自行关键词「Address Generator」搜索），类似这样的：

Random US Street Address

注意点：填写地址时请尽量选择美国的免税州，以防在购买服务时可能带来税费（州消费税 State Sales Tax），经测试，在选非免税州情况下订阅 Applle Music 服务，会被税（支付时不会有提示，但账单上会加入）….美国的免税州主要有以下几个：

  • Oregon 俄勒冈
  • Alaska 阿拉斯加
  • Delaware 特拉华州
  • Montana 蒙大拿
  • New Hampshire 新罕布什尔
  • New Jersey 新泽西。部分免税。此外，California 加利福尼亚州在购买应用时也属于免税地带，其它未知。

非必须准备

想到了就顺带写一下。可能存在需求但并非必须的条件，不感兴趣可略过：

  • 一个美区手机号；

由于转区涉及的地址及联系方式变更是基于订单寄送地址，与 Apple ID 可以绑定的双重验证所使用的手机号是相互独立的；加之，即使是美区 Apple ID 的双重验证也可以选择绑定国内的手机号，因而美区的手机号并非必要条件。国内办理美区手机号没有相应的信息资料及支付手段，当然不现实。但如果需要的话，可以选择以 Google Voice（GV） 所提供的虚拟手机号作为替代品。

目前自行注册 GV 获取个人专属的手机号，流行的新旧方案要么不可行要么可行度很低，快捷的方式还是通过某电商平台出售的绑定 GV 号码的 Gmail 帐号，再通过转移的手段换绑个人的 Gmail 邮箱。而进行相关的设置与转移手续，都必须在全程为美国 IP 地址的网络环境下进行，不然无法正常完成。

操作确认

如何判断个人 Apple ID 帐号的 iCloud 是否在 Apple 国内的合作者云上贵州运行？

  1. 访问「停用 iCloud（中国）」这个页面：http://www.icloud.com/optout

  2. 点击最下方「于 2018 年 2 月 28 日起停用 iCloud」

  3. 在登录个人帐号后，如跳转页面提示如下，则 iCloud 所属 Apple ID 已经安全脱离某区。

如图，提示帐号已完成转区。

应用购买

国区的帐号有银联系银行卡信用卡、微信支付及支付宝支付等多种我们易于获取的支付渠道加持，所以购买各种应用与服务也相对方便。而转区之后，要是没有也没法绑定双币卡，这事就没有那么便捷了。当然，基本的使用并没有问题，只要通过某电商平台渠道购买用于 App Store 或 iTunes Store 的 Gift cards 进行充值，转化为自己 Apple ID 的余额，用于日常的支付。

Gift cards 有 $10、$25、$50、$100、$200 等多种面值可供选购。购买应用一类请选购 App Store 的 Gift cards。

在余额充足的情况下，包括付费应用的购买、iCloud 云存储的订阅、Apple Music 的订阅及应用内购或相关的应用内服务订阅，经测试都可以达成。

需要注意的是，如果要为美区 Apple ID 设置家庭共享群组，无法通过 Apple ID 余额的途径达成，此时支付渠道必须绑定可用的银行卡。
